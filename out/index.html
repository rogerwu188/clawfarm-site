How it works

From request to settlement
in one flow.

01
Register
Providers register models, APIs, GPU nodes, agent services, data services, or evaluators.
02
Discover
Agents and apps discover available services through the provider registry.
03
Execute
Requests and tasks are routed through eco, auto, or premium modes.
04
Verify
Usage proofs, task outputs, and evaluator signals confirm valid work.
05
Settle
USDC flows from user escrow to the provider wallet through smart contracts.
06
Reward
Verified contribution earns protocol rewards based on usage, price, and quality.

Why ClawFarm

Open supply. Unified metering.
Automatic settlement.

Open Supply

Anyone can register as a Provider — GPU operators, cloud instances, API resellers, fine-tuned model hosts, data agents, service agents, and evaluators.

Unified Metering

Every AI Work Unit is metered with proofs. Tokens, tool calls, data retrieval, evaluator checks, and task outputs can all enter the ledger.

Automatic Settlement

Smart contracts split verified payments: 97% to Provider, 3% to Treasury. No invoices. No payment delays.

Non-Custodial

User USDC sits in a program-owned escrow (PDA). No human holds the key. Withdraw anytime.

Rewards and economics

Rewards support verified work
after settlement.

$CLAF rewards reinforce useful protocol activity. Providers earn for verified AI Work Units, demand apps earn for real usage, and the treasury receives a small settlement fee for buyback, maintenance, and resilience.

70%
Supply-side Rewards

Distributed to providers based on verified work, price efficiency, and quality scores.

30%
Demand-side Rewards

Earned by protocol users and demand apps proportional to verified AI work consumption.

3%
Autonomous Treasury

70% Buyback · 20% Core Maintenance · 10% Infra Resilience.

Treasury Allocation Mandate

Every settlement collects a 3% fee into the Treasury PDA. An autonomous on-chain AI agent manages these funds:

  • ↪

    70% Buyback & Burn: Constant deflationary pressure for $CLAF.

  • ↪

    20% Core Maintenance: Security audits and protocol development.

  • ↪

    10% Infra Resilience: Funding Mirror Nodes and IPFS/Arweave storage.
