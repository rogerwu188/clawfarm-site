How it works

From request to settlement
in one flow.

01
Provide
Register your model, GPU, or API endpoint. Set your own price. Stake $CLAF.
02
Route
Requests are routed to the best Provider by eco, auto, or premium mode.
03
Meter
Every token is counted client-side. Both parties sign a usage proof.
04
Settle
Smart contract splits payment: 97% Provider, 3% Treasury. Automatic.
05
Earn
$CLAF tokens distributed every Epoch. More usage × better price × higher quality = more rewards.

Why ClawFarm

Open supply. Unified metering.
Automatic settlement.

Open Supply

Anyone can register as a Provider — GPU operators, cloud instances, API resellers, fine-tuned model hosts. No approval. No whitelist.

Unified Metering

Every request is metered with dual-signature verification. Client counts tokens independently. Both parties sign. No trust required.

Automatic Settlement

Smart contract splits every payment: 97% to Provider, 3% to Treasury. No invoices. No payment delays.

Non-Custodial

User USDC sits in a program-owned escrow (PDA). No human holds the key. Withdraw anytime.

Dual-sided mining

Earn $CLAF by serving
or consuming AI.

The $CLAF token is distributed daily to both sides of the marketplace. Supply compute to earn 70%, or build and consume to earn 30%.

Supply-side Rewards

Autonomous Treasury

70% Buyback · 20% Core Maintenance · 10% Infra Resilience.

Treasury Allocation Mandate

Every settlement collects a 3% fee into the Treasury PDA. An autonomous on-chain AI agent manages these funds:

  • ↪

    70% Buyback & Burn: Constant deflationary pressure for $CLAF.

  • ↪

    20% Core Maintenance: Security audits and protocol development.

  • ↪

    10% Infra Resilience: Funding Mirror Nodes and IPFS/Arweave storage.
